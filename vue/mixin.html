# Mixin 混入

混入 (mixin) 提供了一种非常灵活的方式，来分发 Vue 组件中的可复用功能。一个混入对象可以包含任意组件选项。当组件使用混入对象时，所有混入对象的选项将被“混合”进入该组件本身的选项。

使用mixin 实现复制

import Clipboard from "clipboard";  // 复制插件
import { Message } from 'element-ui'

export const clipboardMixin = {
  data(){
    return {
      clipboard: null
    };
  },
  mounted() {
    this.clipboard = new Clipboard(".js-copy");
    // 复制成功执行的回调
    this.clipboard.on("success", function(data) {
        Message({ showClose: false, message: "已复制", type: "success" });
    }); 
  },
  destroyed() {
    this.clipboard.destroy();
  }
}

组件中使用

import { clipboardMixin } from './mixins';
export default {
  mixins: [clipboardMixin],
}
