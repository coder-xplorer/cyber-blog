# 骨架屏

# 实现方式

主要介绍如何实现动画

骨架屏的动画是好像一条光束由左至右扫过去的，所以我会将背景设定为渐层颜色。加入 background，设定为 linear-gradient()，角度倾斜一点，设定为 100deg；然后由全透明的白色，渐变为半透明的白色，再到全透明的白色。 之后再补上基础的背景颜色，浅灰色，现在就可以看到这条光束了。

那么要怎样进行动画呢，我会先将这个背景的宽度，拉成两倍大小，设定为 200%，高度维持是 100%。这样就可以通过 background-position 控制背景的位移，从而达到扫光的效果。

现在当 background-position-x 设定为 100%，光的部份会在左边，而设定为 0% 的话，光的部份会在右边，所以由大的数字改变到小的数字就可以实现左至右的扫光。 不过这还不够，我们需要左右预留一些缓冲，才可以将光由外面扫入，所以先把改为 180%。

最后加入动画，参考下方代码。

html 部分

<div class="skeleton loading">
  <div class="header"></div>
  <div class="wrapper">
    <div class="left-box"></div>
    <div class="right-box">
      <div></div>
      <div></div>
    </div>
  </div>
  <div class="wrapper">
    <div class="left-box"></div>
    <div class="right-box">
      <div></div>
      <div></div>
    </div>
  </div>
</div>

css 部分

.skeleton {
  width: 300px;
  padding: 10px;
}
.skeleton .header {
  width: 100%;
  height: 50px;
  border-radius: 5px;
}
.wrapper {
  display: flex;
  padding: 10px;
}
.left-box {
  width: 60px;
  height: 60px;
  border-radius: 5px;
}
.right-box {
  width: 100%;
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  margin-left: 10px;
}
.right-box div {
  width: 100%;
  height: 25px;
  border-radius: 5px;
}

/*
  添加背景颜色 使用渐进色
*/
.loading .header,
.loading .left-box,
.loading .right-box div {
  background: linear-gradient(
      100deg,
      rgba(255, 255, 255, 0),
      rgba(255, 255, 255, 0.5),
      rgba(255, 255, 255, 0)
    )
    #ededed;
  background-size: 200% 100%; /* 设置背景宽度200% */
  background-position-x: 180%;
  animation: 1s loading ease-in-out infinite;
}
@keyframes loading {
  to {
    background-position-x: -20%;
  }
}
