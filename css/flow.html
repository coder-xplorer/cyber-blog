# 实现方式

通过 column-count 设置为 4 列 间距为 0 给每一个 Item 一个 padding, 图片设置 width: 100%; height:auto; 即可实现效果

.masonry {
  column-count: 4;
  column-gap: 0;
}

.item {
  padding: 2px;
}

.item img {
  display: block; /* 消除图片间隙 */
  width: 100%;
  height: auto;
}

/* display : flex 实现 */
.masonry {
  display: flex;
  flex-direction: column; /*设置垂直方向排布 */
  flex-wrap: wrap; /* 换行*/
  height: 1000px; /* 设置高度，放不下就换行， 图片的高度不一样，会展示瀑布流效果*/
}
.item {
  width: 25%;
  padding: 2px;
  box-sizing: border-box;
}
.item img {
  display: block;
  width: 100%;
  height: auto;
}
