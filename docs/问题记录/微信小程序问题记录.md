## 1. 代码包太大

在小程序中代码包太大，会导致小程序上传代码失败。

解决办法：在项目中将代码压缩选项都勾选上。
将 `"ignoreUploadUnusedFiles": true,` 添加到 `project.config.json` 中。这个是过滤掉一些未使用的文件。

**这里一定要填加上，我遇到在本地代码检验就是可以的，上传代码就显示代码包太大，加上这个选项后，问题就解决了。主要是把 UI 库中未使用的文件过滤掉了。**

其次，就是代码分包，将代码进行分包
分包用到的插件就下载到分包目录中去，需要在分包目录中创建 package.json 文件。

``` js
 // app.json文件中配置分包
"subPackages": [
    {
      "root": "package-public",
      "name": "public",
      "pages": [
        "pages/about/index",
      ]
    }
  ],
```
